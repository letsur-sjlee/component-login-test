# Typography

디자인 시스템의 타이포그래피는 **Pretendard Variable** 폰트를 기반으로 하며, 용도에 따라 **Display**와 **Text** 계열로 나뉩니다.

## Font Family
- **Display/Body/Sans**: `Pretendard Variable`, `Pretendard`, `-apple-system`, `sans-serif`

## Font Weights
- **Regular**: 400
- **Medium**: 500
- **Semibold**: 600
- **Bold**: 700

---

## Display Series
제목이나 강조가 필요한 큰 영역에 사용됩니다.

<div className="sb-unstyled">
  <table className="w-full text-left border-collapse">
    <thead>
      <tr className="border-b border-gray-200">
        <th className="py-2 px-4 font-semibold text-sm text-gray-600">Token</th>
        <th className="py-2 px-4 font-semibold text-sm text-gray-600">Size / Line Height</th>
        <th className="py-2 px-4 font-semibold text-sm text-gray-600">Preview</th>
      </tr>
    </thead>
    <tbody>
      {Object.entries(fontSize)
        .filter(([key]) => key.startsWith('display'))
        .map(([key, [size, { lineHeight }]]) => (
          <tr key={key} className="border-b border-gray-100">
            <td className="py-4 px-4 align-middle">
              <code className="text-xs bg-gray-50 px-1 py-0.5 rounded text-brand-700">{key}</code>
            </td>
            <td className="py-4 px-4 align-middle text-sm text-gray-500">
              {size} / {lineHeight}
            </td>
            <td className="py-4 px-4 align-middle" style={{ fontSize: size, lineHeight: lineHeight }}>
              The quick brown fox
            </td>
          </tr>
        ))}
    </tbody>
  </table>
</div>

## Usage Guidelines
- 모든 텍스트는 토큰에 정의된 **Line Height**를 준수해야 합니다.
- 자간(Letter Spacing)은 폰트 자체의 최적화된 값을 기본으로 사용합니다.
- 모바일 환경에서는 한 단계 작은 사이즈의 토큰 사용을 고려합니다.
