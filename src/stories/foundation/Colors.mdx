# Colors

디자인 시스템의 색상 체계는 **Primitive Palette**와 **Semantic Tokens**로 구성됩니다.

## Primitive Palette
원시 색상 팔레트는 시스템에서 사용하는 모든 색상의 원천입니다. 직접 사용하기보다는 Semantic Token을 통해 참조하는 것을 권장합니다.

  <ColorItem

    subtitle="Basic colors"
    colors={{

      Black: primitives.base.black,
      Transparent: primitives.base.transparent

  />

    title="Gray"

## Semantic Tokens
시멘틱 토큰은 목적과 의미에 따라 색상을 정의합니다. UI 개발 시에는 항상 이 토큰을 사용해야 합니다.

### Text
텍스트와 인라인 요소에 사용되는 색상입니다.
<ColorPalette>
  <ColorItem
    title="Text Primary"
    subtitle="Main text"
    colors={{
      primary: semanticColors.text.primary,
      'on-brand': semanticColors.text['primary-on-brand']
    }}
  />
  <ColorItem
    title="Text Secondary"
    subtitle="Supportive text"
    colors={{
      secondary: semanticColors.text.secondary,
      'secondary-hover': semanticColors.text['secondary-hover']
    }}
  />
  <ColorItem
    title="Status Text"
    subtitle="Error, Warning, Success"
    colors={{
      error: semanticColors.text['error-primary'],
      warning: semanticColors.text['warning-primary'],
      success: semanticColors.text['success-primary'],
      link: semanticColors.text.link
    }}
  />
</ColorPalette>

### Background
