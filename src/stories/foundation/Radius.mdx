# Radius

컴포넌트의 테두리 둥글기(Border Radius)를 정의하는 시스템입니다.

## Radius Tokens

<div className="sb-unstyled">
  <div className="grid grid-cols-2 md:grid-cols-4 lg:grid-cols-5 gap-4 mt-8">
    {Object.entries(borderRadius).map(([key, value]) => (
      <div key={key} className="flex flex-col items-center gap-3 p-4 border border-gray-100 rounded-xl bg-white shadow-sm">
        <div
          className="w-16 h-16 bg-gray-50 border-2 border-brand-500 flex items-center justify-center relative overflow-hidden"
          style={{ borderRadius: value }}
        >
          {/* 안쪽에 격자 무늬를 넣어 곡률이 더 잘 보이게 함 */}
          <div className="absolute inset-0 opacity-10" style={{ backgroundImage: 'linear-gradient(#00C781 1px, transparent 1px), linear-gradient(90deg, #00C781 1px, transparent 1px)', backgroundSize: '10px 10px' }} />
          <span className="text-[10px] font-bold text-brand-700 z-10">{value}</span>
        </div>
        <div className="text-center">
          <code className="text-[11px] font-bold text-gray-900 block">{key}</code>
        </div>
      </div>
    ))}
  </div>
</div>

## 비교 보기 (Comparison)
각 단계별 곡률 차이를 한눈에 비교할 수 있습니다.

<div className="sb-unstyled mt-4 p-8 bg-gray-900 rounded-2xl flex items-end gap-4 overflow-x-auto">
  {['xs', 'sm', 'md', 'lg', 'xl', '2xl'].map((key) => (
    <div key={key} className="flex flex-col items-center gap-2">
      <div
        className="w-12 h-12 bg-white border-2 border-brand-400"
        style={{ borderRadius: borderRadius[key] }}
      />
      <span className="text-[10px] text-gray-400">{key}</span>
    </div>
  ))}
</div>

## Usage Guidelines
- **Small Elements**: 체크박스, 배지 등 작은 요소에는 `xxs` 또는 `xs`를 사용합니다.
- **Buttons & Inputs**: 기본 UI 요소에는 `sm` 또는 `md`를 주로 사용합니다.
- **Cards & Modals**: 큰 컨테이너에는 `lg` 이상의 토큰을 사용합니다.
- **Circular**: 완전한 원형이 필요한 경우 `full`을 사용합니다.
